# 🥫 Board Sauce
**The secret sauce for your table.**

Board Sauce is a suite of mobile-first, high-utility digital companions designed to eliminate the "friction" of analog gaming. No more manual math, lost score-pads, or rule disputes—just pure gameplay.

[Link to Live Site: https://boardsauce.app](https://boardsauce.app)

---

## 🚀 The Philosophy: Utility over Polish
As a Product Designer, I built Board Sauce to solve a specific problem: game-night math is a tax on fun. This project explores **AI-assisted rapid prototyping** to bridge the gap between a "what if" idea and a functional tool used by real players in minutes.

### Key Features:
* **Mobile-First UX:** Designed for one-handed use in social settings. No zooming, no lag, just fast tapping.
* **Standalone Experience:** Optimized for iOS "Add to Home Screen" to feel like a native app.
* **Persistent State:** Uses LocalStorage so your game night isn't ruined by an accidental browser refresh.
* **No Limits:** Whether it's 2 players or 20, the UI adapts to the group.

## 🛠️ Tech Stack & Workflow
* **Frontend:** HTML5, CSS3, Vanilla JavaScript.
* **AI Prototyping:** Built using **Claude Code** and **Gemini** for rapid iteration and logic drafting.
* **Hosting:** GitHub Pages with a custom domain on **Porkbun**.
* **Branding:** Typography-focused UI using `Bebas Neue` for that bold, "Board AF" aesthetic.

## 📸 For the Designer Portfolio
If you're looking at this from a design perspective:
